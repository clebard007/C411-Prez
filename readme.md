# C411 NFO Generator
Prérequis:
- Installer la dernière version de Python
- `git clone https://github.com/clebard007/C411-Prez.git`
- Rensigner votre clef API TMDB
- Enjoy.

## Infos

Le script a été testé sur Windows 11, Ubuntu 22.04 et Macos

## Utilisation

Pour générer un nfo:
`python generate_nfo.py --video_file C:\temp\C411\C411-Prez\film.mkv --output film.nfo`

Pour générer une présentation
`python generate_prez.py`
